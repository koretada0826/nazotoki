# 地頭ジム 🧠

謎解き・計算・思考の型・記憶の4本柱で「地頭」を鍛える、毎日のトレーニングアプリ。
ビルド不要・インストール不要・完全オフライン対応の **Webアプリ（PWA）**。

---

## 使い方（Macで開く）

ターミナルでこのフォルダに入り、サーバーを起動するだけ。

```bash
cd "/Users/koretada/Desktop/地頭"
python3 -m http.server 8731
```

ブラウザで開く → **http://localhost:8731/**

止めるときはターミナルで `Ctrl + C`。

---

## iPhoneで「アプリ」として使う

iPhoneとMacが**同じWi-Fi**につながっていれば、iPhoneのSafariから使えます。

1. Macで自分のIPアドレスを調べる
   ```bash
   ipconfig getifaddr en0
   ```
   （例: `192.168.1.23` のような数字が出る）
2. 上のサーバーを起動したまま、iPhoneのSafariで
   `http://（その数字）:8731/` を開く
3. 共有ボタン → **「ホーム画面に追加」**
   → アイコンがホームに出て、本物のアプリのように全画面で起動します。

> ※「いつでもどこでも・Macを起動しなくても使いたい」場合は、
> 無料の静的ホスティング（Netlify / GitHub Pages / Cloudflare Pages）に
> このフォルダを置けばOK。次のステップでやりましょう。

## 中身（4本柱）

| 柱 | ドリル |
|----|--------|
| 🧩 論理・推理 | 数列推理 / うそつき推理 / 順序推理 |
| 🔢 計算・数的処理 | 暗算スピード / 概算力 / 割合・損益 |
| 🧠 思考の型 | MECE・仮説・期待値・サンクコストなど16問の選択式 |
| ⚡ 記憶・WM | 数字スパン / Nバック |

- 問題は**毎回自動生成**（思考の型のみ良質な手作り問題バンク）。
- 正解率と速さから**難易度が自動調整**（適応エンジン）。
- 進捗は**この端末のブラウザ内に保存**（XP・レベル・連続日数）。

## ファイル構成

```
index.html          画面の入れ物
styles.css          デザイン
app.js              画面遷移・出題・採点の本体
manifest.webmanifest  アプリ情報（PWA）
sw.js               オフライン用キャッシュ
icon.svg            アイコン
src/
  util.js           小道具（乱数・DOM生成など）
  store.js          進捗の保存（localStorage）
  engine.js         難易度の自動調整・XP計算
  drills/
    index.js        ドリル登録
    logic.js        論理・推理
    calc.js         計算・数的処理
    framework.js    思考の型（問題バンク）
    memory.js       記憶・WM
```
