# ナゾトキ 🗝️

ひらめき・暗号・推理の謎を解いて、**解くスピードと解き方のテクニック**を鍛える謎解き特化アプリ。
ビルド不要・インストール不要・オフライン対応の Webアプリ（PWA）。

---

## 起動（Macで開く）

ターミナルでこのフォルダに入り、サーバーを起動：

```bash
cd "/Users/koretada/Desktop/地頭"
python3 server.py
```

ブラウザで **http://localhost:8755/** を開く。
止めるときはターミナルで `Ctrl + C`。

> `server.py` はキャッシュ無効化つき。コードを直したら、ブラウザを普通に再読み込みするだけで反映されます。

---

## iPhoneで「アプリ」として使う

iPhoneとMacが同じWi-Fiなら、iPhoneのSafariから使えます。

1. MacのIPを調べる： `ipconfig getifaddr en0` （例: `192.168.1.23`）
2. iPhoneのSafariで `http://（そのIP）:8755/` を開く
3. 共有ボタン →「ホーム画面に追加」→ 本物のアプリのように全画面起動

## 中身

| 区分 | 内容 |
|------|------|
| 💡 ひらめき | 手作りの良問12問（なぞなぞ・発想・ラテラル）。段階ヒント＆解説つき |
| 🔐 暗号解読 | 自動生成で無限（逆さ・A1Z26・シーザー・五十音ずらし・記号対応） |
| 🔍 推理・脱出 | 金庫の暗証番号を自動生成（答えは必ず1つに決まる）＋名作3問 |
| ⚡ スピード謎解き | 60秒タイムアタック。解く速さを鍛える。自己ベスト記録 |
| 🎓 解き方テクニック | 暗号の崩し方・ひらめきの視点・推理の整理術など5本 |

- 進捗（解いた数・ノーヒント・連続日数・速解き最高）はこの端末のブラウザに保存。
- 全問に**段階ヒント**と**答え・解説**。詰まっても"型"を学べる。

## ファイル構成

```
index.html / styles.css / app.js   画面・デザイン・本体
server.py                          ローカルサーバー（キャッシュ無効）
manifest.webmanifest / sw.js / icon.svg   PWA設定・オフライン・アイコン
src/
  util.js          小道具（乱数・DOM生成・答え合わせの正規化）
  store.js         進捗の保存（localStorage）
  techniques.js    解き方テクニックの教材
  puzzles/
    index.js       カテゴリ登録
    hirameki.js    ひらめき（手作り）
    cipher.js      暗号（自動生成）
    deduction.js   推理・脱出（金庫=自動生成＋名作）
legacy_jigatama/   旧「地頭ジム」（4本柱版）を保存。/legacy_jigatama/ で開ける
```
